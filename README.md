# Tiny Twitter 

This  application  provides some features of Twitter Website.
- Registration 
- Tweet
- Report of user and tweets


And it has 2 main packages
- JWT
- PDF

# How to run
 you can simply install composer
```shell script
composer install
``` 
Then
```shell script
php artisan serve
```
Then migrate datebase
```shell script
php artisan migrate
```
Then seed Datebase
```shell script
php artisan db:seed
```
